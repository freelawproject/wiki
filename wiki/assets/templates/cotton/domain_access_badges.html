{% comment %} Domain-access badges — favicons of the outside domains granted access to a page/directory (including via an ancestor directory). Staff-only: the views set `access_domains` only for staff, so this renders nothing for everyone else. Each badge shows the domain's favicon (or a fallback glyph) and a tooltip naming the domain. Usage: {% endcomment %} {% if domains %} {% for d in domains|slice:":5" %} {% if d.has_favicon %} {% else %} {% endif %} {% endfor %} {% if domains|length > 5 %} +{{ domains|length|add:'-5' }} {% endif %} {% endif %}
